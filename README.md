# Digicloudify OS Monorepo

## Workspace Structure

- `frontend` - Next.js marketing UI
- `backend` - NestJS API service
- `workers` - background job processors
- `shared` - common utilities and shared code
- `scripts` - repository utility scripts

## Quick Start

```bash
npm install
npm run check-env
```

Run services:

```bash
npm run dev:frontend
npm run dev:backend
npm run dev:workers
```
